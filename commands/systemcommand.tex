\section{systemcommand}
\label{cmd:systemcommand}

\SACTitle{概要}
从SAC中执行系统命令

\SACTitle{语法}
SystemCommand command [ options ]

\SACTitle{输入}
\begin{itemize}
\item command : 系统命令名 
\item options : 命令需要的选项 
\end{itemize}

\SACTitle{说明}
这个命令允许你在执行SAC的同时执行系统命令
