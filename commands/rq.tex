\SACCMD{rq}
\label{cmd:rq}

\SACTitle{概要}
去除谱文件中的地震Q因子

\SACTitle{语法}
RQ [ Q v ] [ R v ] [ C v ]

\SACTitle{输入}
\begin{itemize}
\item Q v : 设置质量因子为v 
\item R v : 设置距离为v，单位为km 
\item C v : 设置群速度为v，单位km/s 
\end{itemize}

\SACTitle{缺省值}
RQ Q 1. R 0. C 1.

\SACTitle{说明}
用于校正振幅的方程如下:
\[ AMP_{corrected}(f) = AMP_{uncorrected}(f) * e^{\frac{\pi R F}{Q C}} \]
其中f为频率，单位为Hz，R为距离，单位km，C是群速度，单位为km/s。Q是一个无量纲质量因子

\SACTitle{头段变量改变}
DEPMIN, DEPMAX, DEPMEN

\SACTitle{错误消息}
\begin{itemize}
\item[-]1301: 未读入数据文件
\item[-]1305: 对时间序列非法操作
\end{itemize}
