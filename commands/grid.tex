\SACCMD{grid}
\label{cmd:grid}

\SACTitle{概要}
控制绘图时的网格线

\SACTitle{语法}
\begin{SACSTX}
GRID [ON|OFF|S!OLID!|D!OTTED!]
\end{SACSTX}

\SACTitle{输入}
\begin{itemize}
\item ON : 打开网格绘图但不改变网格类型
\item OFF : 关闭网格绘图 
\item SOLID : 打开网格绘图并使用实网格线 
\item DOTTED : 打开网格绘图并使用虚线绘制网格线 
\end{itemize}

\SACTitle{缺省值}
\begin{SACDFT}
GRID OFF
\end{SACDFT}

\SACTitle{说明}
这个命令控制X和Y两个坐标轴，XGRID和YGRID分别控制单个坐标轴的网格类型.

\SACTitle{相关命令}
\nameref{cmd:xgrid}、\nameref{cmd:ygrid}
