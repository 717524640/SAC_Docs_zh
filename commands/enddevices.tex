\section{enddevices}
\label{cmd:enddevices}

\SACTitle{概要}
结束一个或两个图形设备

\SACTitle{语法}
BeginDevices Sgf | Xwindows

\SACTitle{其他形式}
ENDG以及EG也是这个命令可接受的名字

\SACTitle{输入}
\begin{itemize}
\item Sgf : SAC图形文件设备驱动
\item Xwindows : X-windows窗口显示系统
\end{itemize}

\SACTitle{说明}
这个命令终止一或两个图形设备。设备用BEGINDEVICES激活

\SACTitle{相关命令}
BEGINDEVICES
